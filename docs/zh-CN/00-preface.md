## 贡献方式

+ Fork 这个项目
+ 请不要直接在 `README.md` 中直接添加内容
+ 所有的文档都放在 `docs` 中, 请根据内容找到相应的文件并添加
+ 如果是 Mac 或者 Linux 用户, 请在提交前运行 ./build.sh 来自动生成 `README.md` 文件
+ 保存并提交
+ 新建一个 Pull Request
